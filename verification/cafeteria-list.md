# 성균관대 명륜캠(인문사회과학캠퍼스) 학생식당 목록 — 공식 페이지 대조

**요약 (3줄)**
- 공식 학생식당 목록 페이지(`welfare_11.do`)에는 명륜캠 식당이 **4곳만** 등록돼 있다. 옥류천은 이 목록에 없다.
- **옥류천은 학생식당이 아니라 "교수식당"이다.** 위치는 경영관·학술정보관 사이 **교수회관 1층** — 600주년기념관(패컬티식당)과는 별개 건물. 원래 "교수식당"으로 불리다 교내 이름 공모전에서 "옥류천"이라는 이름을 얻었다 (성균웹진, 접근 2026-07-24). `welfare_11.do` HTML에 `옥류천` 문자열 자체가 없고, `data-con` 코드도 없다 — 학생식당 시스템 대상이 아니라서 크롤러가 못 잡는 게 정상이다.
- 현재 등록된 4곳(10201030·31·33·34)은 **전부 공식 목록과 정확히 일치**한다. 빠진 코드·새 코드 없음.

## 조사 방법

1. `curl -s -A "Mozilla/5.0 ..." https://www.skku.edu/skku/campus/support/welfare_11.do` 로 명륜캠 학생식당 목록 페이지(mode 파라미터 없음 = 기본 목록) HTML 원본을 그대로 받았다.
2. `cafeteriaBtn`, `data-con` 문자열을 grep — `<li>` 블록마다 `<h5>식당명</h5>`, `<dd>건물/층</dd>`, `data-con="코드"` 가 1:1로 대응하는 것을 확인했다 (라인 3248~3403).
3. 결과: `data-con` 값은 `10201030`(패컬티), `10201031`(은행골), `10201034`(법고을), `10201033`(금잔디) **4개뿐**. 페이지 전체에 `옥류천` 문자열 검색 결과 **0건**.
4. 명륜캠/자연캠 판별 근거: 페이지 상단에 캠퍼스 탭이 있고 `자연과학캠퍼스` 링크가 `/skku/campus/support/welfare_11_1.do` 로 별도 존재함(라인 3185) — 즉 `welfare_11.do` 자체가 명륜캠 전용 페이지. 건물명(600주년기념관·법학관·경영관)도 전부 명륜캠 소속으로 교차 확인됨.
5. 옥류천 관련 정보가 학생식당 페이지에 없어, 보조 확인으로 WebSearch → 성균웹진 기사(`culture04.do?articleNo=74725`)를 열어 옥류천의 정체를 확인했다: **교수식당**이며 위치는 **교수회관 1층**(경영관·학술정보관 사이) — 패컬티식당(600주년기념관 6층)과는 별개 건물·별개 시설이다. 가격대(8,000~12,000원, 학생식당의 약 2배)와 주 이용객(교수)도 명시돼 있어, 학생식당 크롤링 대상에서 원래 제외되는 게 맞다는 근거가 된다.
6. `conspaceCd`를 옥류천에 대해 별도로 찾으려 했으나, **학생식당 목록 페이지 자체에 옥류천 항목이 없어 코드 추출이 불가능했다** — 추측 코드는 적지 않는다("코드 미확인"). 옥류천이 이 URL 스킴으로 메뉴를 제공하는지 여부도 이번 조사로는 확인 못 했다(공식 학생식당 목록에 없다는 사실만 확정).

## 결론

- **현재 등록된 4곳의 코드는 전부 유효하고 정확하다.** 수정 불필요.
- **옥류천은 이 서비스가 다루는 "학생식당" 범주에 속하지 않는다** (교수식당, 별도 건물, 별도 가격대). 크롤러 목록에 추가할 근거가 약하다 — 추가하려면 옥류천이 실제로 `welfare_11.do`류 시스템으로 메뉴를 게시하는지부터 확인해야 하는데, 이번 조사에서는 그 문서를 찾지 못했다.
