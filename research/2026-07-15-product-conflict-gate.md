## 2. 종합 판정: KILL

거지맵이 결정타 — 2026-03-20 출시, 누적 이용자 131만(웹+앱). 팀 가설의 3축(예산 필터 + 학식 포함 + 대학가 지도)을 이미 전국 단위로 제공하고 시장 검증까지 끝냄. "범용 LLM 대비 차별성" 이전에 "기존 제품 대비 차별성"에서 막힘. 학식+지도 조합은 대학마다 학생 사이드프로젝트로 반복 제작됨 — "안 풀린 문제"가 아니라 "여러 번 풀렸고 하나는 전국구로 성공한 문제".

## 3. 범용 LLM이 못 하는 것 (차별화 가능 지대)

1. 오늘 날짜 기준 실시간 학식 메뉴를 모른다.
2. 실제 현재 가격을 모른다 — 숫자류는 환각 취약 영역.
3. 영업 여부·브레이크타임·폐업을 실시간 반영 못한다.
4. 위치 기반 거리 계산을 GPS 기준으로 못한다.
5. **개인 취향·식사 히스토리를 축적해 반영하지 못한다 (상태 없음).**
6. 크라우드 실시간 제보(마감세일·웨이팅)를 못 담는다.

단, 거지맵이 1·2·6을 상당 부분 커버 — **LLM이 못 하는 것 ∩ 거지맵이 못 하는 것**만이 진짜 빈틈.

## 4. 관측된 빈틈

- 성대 공식 채널의 메뉴·가격 콘텐츠 부실 (직접 확인) — 단 서드파티가 이미 메워온 유형의 빈틈.
- 거지맵의 성대 상권(명륜·율전) 데이터 밀도 미검증 — 유일하게 관측된 검증 가능한 빈틈이나, 이는 커버리지 문제지 문제 정의의 새로움이 아님.
- **가격 필터+학식+지역+대학생 조합 자체에서 새로 주장할 미해결 영역은 확인되지 않음.**

- 망고플레이트 2023-10-31 종료 확인. 사인은 1차 근거 미확보(추정 금지).
- 학식 앱은 죽기보다 **대학마다 저비용으로 반복 재생산되며 살아있음** — "시장이 없다"가 아니라 "진입장벽이 없어 차별화 포인트가 될 수 없다"는 위험 신호.
