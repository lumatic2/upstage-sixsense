## 2. 파편화 실태 (관측 사실)

- **성균관대 학식 자체는 "파편화" 프레임이 안 맞는다.** 인사캠 식단은 깔끔한 HTML 표 + 텍스트 가격 (스크린샷 확인). 서울대도 동일.
- 다만 캠퍼스별·식당별 URL이 따로고(인사캠/자연캠/기숙사는 다른 서브도메인), JS 팝업·쿼리 기반이라 **단순 GET으로는 못 긁고 브라우저 렌더링 필요** — "흩어짐"이라기보다 "동적 렌더링" 문제.
- **진짜 파편화는 학교 주변 식당**: 네이버 Local API에 가격 필드가 없고, 착한가격업소 CSV는 극소수. **주변 식당 실제 메뉴 가격은 구조화 데이터로 거의 존재하지 않으며, 메뉴판 사진·블로그 리뷰 사진 안에 갇혀 있을 가능성이 높다** — 단 실제 메뉴판 사진을 열어 확인하지는 못함(후속 실사 필요, 미검증 가설).

## 3. Document AI 필요/불필요 지점

- **불필요**: 성대 인사캠 학식(S1)·서울대(S4) — 이미 구조화된 HTML. 여기 Document Parse를 쓰면 "요건 채우려는 억지" 모양.
- **필요 가능성**: 주변 식당 메뉴판 사진, 전단, (있다면) HWP 공문 — 단 미검증.

## 4. Upstage 제품 사실 (공식 문서 확인)

- Document Parse 입력: JPEG, PNG, BMP, PDF, TIFF, HEIC, DOCX, PPTX, XLSX, **HWP, HWPX** (네이티브 지원 확인). 50MB, 동기 100p/비동기 1,000p.
- 요금: Document Parse Standard $0.01/p, Enhanced $0.03/p. Solar Pro 3/Pro 2: in $0.15/1M, out $0.6/1M.
- 무료: 가입 시 $10 크레딧, Upstage Studio 에이전트당 10회.
- 교육기관: "Upstage x AWS AI Initiative" — 학교 대상 Solar Pro·Document Parse 최대 1년 무료.
